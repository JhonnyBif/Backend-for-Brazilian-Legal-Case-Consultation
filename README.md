# PTBR

## Backend para Consulta de Processos Jurídicos Brasileiros

Este repositório contém o código-fonte para o **backend** desenvolvido para um projeto acadêmico com o objetivo de permitir consultas a processos jurídicos brasileiros utilizando o número do processo como parâmetro de busca.

## Tecnologias Utilizadas

- **Nest.js**

## Pré-requisitos

Antes de começar, certifique-se de ter instalado em sua máquina:

- **Node.js**
- **npm** (ou yarn)

## Instalação

1. **Clone este repositório:** `git clone https://github.com/JhonnyBif/Backend-for-Brazilian-Legal-Case-Consultation`
2. **Navegue até o diretório do projeto:** `cd Backend-for-Brazilian-Legal-Case-Consultation`
3. **Instale as dependências:** `npm install`

## Uso

1. **Inicie o servidor:** `npm start`
2. **Acesse** `http://localhost:3000` **no seu navegador.**
3. **Envie uma requisição GET para** `http://localhost:3000/case_number` **com um número de processo a ser consultado.**

## Acessando o Frontend

Para acessar o repositório do frontend, por favor visite o [Repositório do Frontend](https://github.com/JoaoVitorAlvesR/Frontend-for-Brazilian-Legal-Case-Consultation).

## Descrição

Repositório inicial do framework [Nest](https://github.com/nestjs/nest) em TypeScript.

## Instalação

```bash
$ npm install
```

## Executando o aplicativo

```bash
# desenvolvimento
$ npm run start

# modo watch
$ npm run start:dev

# modo produção
$ npm run start:prod
```

## Testes

```bash
# testes unitários
$ npm run test

# testes e2e
$ npm run test:e2e

# cobertura de testes
$ npm run test:cov
```

# EN

This repository contains the source code for the **backend** developed for an academic project aimed at allowing consultations to Brazilian legal cases using the case number as a search parameter.

## Technologies Used

- **Nest.js**

## Prerequisites

Before getting started, make sure you have installed on your machine:

- **Node.js**
- **npm** (or yarn)

## Installation

1. **Clone this repository:** `git clone https://github.com/JhonnyBif/Backend-for-Brazilian-Legal-Case-Consultation`
2. **Navigate to the project directory:** `cd Backend-for-Brazilian-Legal-Case-Consultation`
3. **Install the dependencies:** `npm install`

## Usage

1. **Start the server:** `npm start`
2. **Access** `http://localhost:3000` **in your browser.**
3. **Send a GET request to** `http://localhost:3000/case_number` **with a case number to be queried.**

## Accessing the Frontend

To access the frontend repository, please visit the [Frontend Repository](https://github.com/JoaoVitorAlvesR/Frontend-for-Brazilian-Legal-Case-Consultation).
